new SpacialIndex(label, cellWidth, cellHeight, cellRatio)

Class to manage a logical 2D grid of cells for registering basic rectangle bounding boxes to query hit/click detection.

Parameters:
Name Type Description
label string

the unique label for this spacial index

width of each cell

Methods

add(entity)

Source:
  • classes/SpacialIndex.js, line 51

Register the entity, registers to each cell it lies on. Requires an upperLeft/lowerRight bound set.

Example
let entity = {
    id: 'vertex1',
    upperLeft: {x: 0, y: 0},
    lowerRight: {x: 10, y: 10}
};
//
spacialIndex.add(entity);
Parameters:
Name Type Description
entity object

the object to register
Throws:

if entity given is missing upperLeft/lowerRight bounding box properties
